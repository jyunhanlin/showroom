# Showroom

Generate GitHub Pages from branch

- [r3f-playground](https://jyunhanlin.github.io/showroom/r3f-playground/)

## Setup and use

### Install

```sh
npm install
```

### Development

```sh
npm run dev
```

## Tech stack

- Next.js
- React.js
- Chakra-ui
